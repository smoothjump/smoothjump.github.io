Hexo 是一个使用node.js 开发的静态博客页面生成工具，Github 支持注册用户使用类似username.github.io 的名称的仓库直接进行该域名的跳转和访问。

该文章省略了markdown/yaml文件语法及Github 详细使用步骤

以下是简单的操作步骤:

  1. 注册Github账号，然后创建仓库(repository)，这里本人的用户名为smoothjump，所以创建一个名为smoothjump.github.io 的仓库。
  2. 在github上传一个简单的readme.md和index.html，前者用于介绍，后者用于跳转博客首页。这里也可以使用命令行提交文件，推送主分支。

做完这些之后就可以直接在浏览器访问地址: https://smoothjump.github.io
页面出现Hello World 字样。

  1. 安装Hexo

安装之后执行博客的初始化: 

hexo init smoothjump.github.io
cd smoothjump.github.io
npm install

缺失模块。
1、请确保node版本大于6.2
2、在博客根目录（注意不是archer根目录）执行以下命令：
npm i hexo-generator-json-content --save
3、在根目录_config.yml里添加配置： 
jsonContent:
  meta: false
  pages: false
  posts:
    title: true
    date: true
    path: true
    text: false
    raw: false
    content: false
    slug: false
    updated: false
    comments: false
    link: false
    permalink: false
    excerpt: false
    categories: true
    tags: true
